# Half Mile Cafe ☕

A modern and responsive cafe website built using **React.js** and **Vite**.
This project showcases a premium coffee shop landing page with elegant UI, smooth layout sections, and a clean user experience.

---

## 🚀 Features

* Modern Hero Section
* Responsive Navbar
* Beautiful Gallery Layout
* Menu Showcase
* Testimonials Section
* Reservation Section
* Location & Footer
* Fully Responsive Design
* Fast Performance with Vite

---

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript
* CSS3

---

```bash
src/

 ├── main.jsx
```

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/saha007subham/half-mile-cafe.git
```

Navigate to the project folder:

```bash
cd half-mile-cafe
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
